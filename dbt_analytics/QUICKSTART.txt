QUICK START
===========

1. Install dbt
   pip install dbt-bigquery

2. Set up credentials
   - Copy profiles.yml to ~/.dbt/profiles.yml (Mac/Linux)
   - OR %USERPROFILE%\.dbt\profiles.yml (Windows)
   - Update keyfile path if needed (defaults to /opt/airflow/service-account.json)

3. Run dbt
   dbt run          # Create all models
   dbt test         # Run data quality tests
   dbt docs generate  # Create documentation

4. View results
   - Check BigQuery for new datasets: dbt_analytics_dev or dbt_analytics_prod
   - View docs: dbt docs serve (opens at localhost:8000)
